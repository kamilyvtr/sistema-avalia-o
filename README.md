## Sobre o Projeto

Criado durante os estudos de **banco de dados** e **PHP**, este sistema permite que usuários deixem avaliações de forma simples. Apesar de todos os arquivos serem `.php`, cada um deles **mistura**:

- **HTML** para estrutura das páginas.
- **Tailwind CSS** (via CDN) para estilização rápida e responsiva.
- **JavaScript** para pequenas interações e validações.
- **PHP** para lógica de servidor, conexão com banco e persistência dos dados.

---

## Tecnologias Utilizadas

| Tecnologia | Como é usada |
|------------|---------------|
| **PHP** | Lógica de backend, conexão com banco de dados, manipulação de formulários. |
| **HTML5** | Estrutura das páginas (embutida nos arquivos `.php`). |
| **Tailwind CSS** | CDN incluso no `<head>` – fornece utilitários CSS responsivos sem arquivo próprio. |
| **JavaScript** | Validações de formulário, controle de estrelas (se houver), feedback visual. |
| **MySQL / MariaDB** | Armazenamento das avaliações. |
